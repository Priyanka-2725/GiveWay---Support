# GiveWay - NGO Donation Platform

A Next.js application for connecting donors with NGOs, built and run entirely on the local system with client-side persistence for demo data and authentication.

## Tech Stack

- **Frontend:** Next.js 15, React 19, TailwindCSS
- **Data:** Client-side local storage and seeded mock content
- **Authentication:** Local-only auth flow

## Getting Started

To run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) to view the application.
